Performance Module

Performance evaluation module for incentive extraction from job advertisements.

This module calculates accuracy and recall metrics by comparing extracted job data against a validation dataset. It processes incentive and non-incentive columns separately and generates both overall and per-column performance metrics.

The results are printed to console and saved to a CSV file for historical tracking.
