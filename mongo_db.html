MongoDB Module

Mongo-DB Utility Module

This module provides functionality for connecting to Mongo-DB, retrieving documents, and converting them to json format compatible with the existing pipeline. It includes functions for establishing database connections, yielding job documents from collections, and transforming Mongo-DB bson documents into standardized json structures.

The module supports various job board formats with special handling for portal-specific fields (indeed or stepstone) and ObjectId conversion. It also includes a test function to verify connection and document conversion functionality.
